# <a name="month-item-recurrence"></a>Month (Item Recurrence)

El **elemento Month** describe el mes en el que se produce un elemento periódico anual. 
  
```xml
<Month/>
```

 **MonthNamesType**
## <a name="attributes-and-elements"></a>Atributos y elementos

En las siguientes secciones se describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

|**Elemento**|**Descripción**|
|:-----|:-----|
|[AbsoluteYearlyRecurrence](absoluteyearlyrecurrence.md) <br/> |Representa un patrón de periodicidad anual.  <br/> |
|[RelativeYearlyRecurrence](relativeyearlyrecurrence.md) <br/> |Describe un patrón de periodicidad anual relativo.  <br/> |
   
## <a name="text-value"></a>Valor de texto

Se requiere un valor de texto. Los siguientes son los valores de texto posibles para este elemento:
  
- Enero
    
- Febrero
    
- Marzo
    
- Abril
    
- Mayo
    
- Junio
    
- Julio
    
- Agosto
    
- Septiembre
    
- Octubre
    
- Noviembre
    
- Diciembre
    
## <a name="remarks"></a>Comentarios

El esquema que describe este elemento se encuentra en el directorio virtual EWS del equipo que ejecuta MicrosoftExchange Server 2007 que tiene instalado el rol de servidor Acceso de cliente.

# <a name="numberofmemberswithconflict"></a>NumberOfMembersWithConflict

El **elemento NumberOfMembersWithConflict** representa el número de miembros de la lista de distribución que tienen un conflicto con una hora de reunión sugerida. Este elemento representa los miembros que tienen un estado **de Ocupado,** **OOF** o **Provisional.**

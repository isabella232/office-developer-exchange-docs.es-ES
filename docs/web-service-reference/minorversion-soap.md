# <a name="minorversion-soap"></a>MinorVersion (SOAP)

El **elemento MinorVersion** representa el número de versión secundaria del servidor. 
  
```XML
<MinorVersion/>
```

 **int**
## <a name="attributes-and-elements"></a>Atributos y elementos

En las siguientes secciones se describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

|**Elemento**|**Descripción**|
|:-----|:-----|
|[ServerVersionInfo (SOAP)](serverversioninfo-soap.md) <br/> |Contiene la versión del servidor que procesó la solicitud.  <br/> |
   
## <a name="text-value"></a>Valor de texto

Valor entero que representa el número de versión secundaria del servidor que procesó la solicitud.

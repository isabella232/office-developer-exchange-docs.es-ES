# <a name="arrayofdomainresponse-soap"></a>ArrayOfDomainResponse (SOAP)

El **elemento ArrayOfDomainResponse** contiene una matriz de respuestas para la configuración de cada dominio solicitado. 
  
```XML
<ArrayOfDomainResponse>
   <DomainResponse/>
</ArrayOfDomainResponse>
```

 **ArrayOfDomainResponse**
## <a name="attributes-and-elements"></a>Atributos y elementos

En las siguientes secciones se describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

|**Elemento**|**Descripción**|
|:-----|:-----|
|[DomainResponse (SOAP)](domainresponse-soap.md) <br/> |Contiene la configuración solicitada para el dominio especificado.  <br/> |
   
### <a name="parent-elements"></a>Elementos principales

Ninguno.

# <a name="meetingrequestwassent"></a>MeetingRequestWasSent

El **elemento MeetingRequestWasSent** indica si se ha enviado una solicitud de reunión a los asistentes solicitados. 
  
```xml
<MeetingRequestWasSent/>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Atributos y elementos

En las siguientes secciones se describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

|**Elemento**|**Descripción**|
|:-----|:-----|
|[CalendarItem](calendaritem.md) <br/> |Representa un Exchange de calendario.  <br/> |
|[MeetingRequest](meetingrequest.md) <br/> |Representa una solicitud de reunión en Exchange almacén.  <br/> |
   
## <a name="text-value"></a>Valor de texto

Si se incluye este elemento, se requiere un valor de texto que represente un valor booleano. Un valor de **true** indica que se envió una solicitud de reunión. Un valor de **false** indica que no se ha enviado una solicitud de reunión. 
  
## <a name="remarks"></a>Comentarios

El esquema que describe este elemento se encuentra en el directorio virtual EWS del equipo que ejecuta MicrosoftExchange Server 2007 que tiene instalado el rol de servidor Acceso de cliente.

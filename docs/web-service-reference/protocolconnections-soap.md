# <a name="protocolconnections-soap"></a>ProtocolConnections (SOAP)

El **elemento ProtocolConnections** contiene cero o más conexiones de protocolo. 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```

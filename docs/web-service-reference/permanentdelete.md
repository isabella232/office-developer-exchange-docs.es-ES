# <a name="permanentdelete"></a>PermanentDelete

El **elemento PermanentDelete** indica si los mensajes se eliminarán permanentemente y no se guardarán en la carpeta Elementos eliminados. 
  
```XML
<PermanentDelete>true | false</PermanentDelete>
```

 **Boolean**
## <a name="attributes-and-elements"></a>Atributos y elementos

En las siguientes secciones se describen los atributos, elementos secundarios y elementos primarios.
  
### <a name="attributes"></a>Atributos

Ninguno.
  
### <a name="child-elements"></a>Elementos secundarios

Ninguno.
  
### <a name="parent-elements"></a>Elementos principales

|**Elemento**|**Descripción**|
|:-----|:-----|
|[Actions](actions.md) <br/> |Representa el conjunto de acciones que están disponibles para realizarse en un mensaje cuando se cumplan las condiciones.  <br/> |
   
## <a name="text-value"></a>Valor de texto

Un valor de texto **de true** indica que el mensaje debe marcarse para eliminarse permanentemente. Un valor de **false** indica que el mensaje no debe marcarse para eliminarse permanentemente. 
  
## <a name="remarks"></a>Comentarios

El esquema que describe este elemento se encuentra en el directorio virtual IIS que hospeda los servicios Web Exchange.

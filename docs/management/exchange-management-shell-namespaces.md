# <a name="exchange-management-shell-namespaces"></a>Espacios de nombres del Shell de administración de Exchange

Busque información sobre los espacios de nombres del Shell Exchange administración en Exchange.
  
**Se aplica a:** Exchange Online | Exchange Server 2013 | Office 365
  
Exchange Las aplicaciones del Shell de administración para Exchange Online, Exchange Online como parte de Office 365 o una versión de Exchange a partir de Exchange 2013 interactúan con el entorno Windows PowerShell a través de los tipos expuestos por el **Espacio de nombres System.Management.Automation.** Los cmdlets Exchange Shell de administración usan tipos de varias Exchange bibliotecas. En esta sección se proporciona información de referencia sobre los tipos e interfaces de las bibliotecas de Exchange que usan los cmdlets Exchange Shell de administración.
